# 🚗 AI Car Marketplace\n\nAI-powered car marketplace built using modern full-stack technologies.\n\n---\n\n## 🔥 Features\n\n* 🤖 AI-based car recommendations\n* 🔐 Secure authentication (Clerk)\n* 📊 Interactive dashboard\n* ⚡ Fast performance with Next.js\n* 🛡️ Backend security with ArcJet\n\n---\n\n## 🛠️ Tech Stack\n\n* **Frontend:** Next.js, Tailwind CSS\n* **Backend:** Prisma, Supabase\n* **Authentication:** Clerk\n* **AI Integration:** Gemini API\n* **Security:** ArcJet\n\n---\n\n## ⚙️ Installation\n\n```bash\ngit clone https://github.com/satyavhan1/AI-Car-Marketplace.git\ncd AI-Car-Marketplace\nnpm install\nnpm run dev\n```\n\n---\n\n## 🔑 Environment Variables\n\nCreate a `.env` file:\n\n```env\nDATABASE_URL=\nDIRECT_URL=\n\nNEXT_PUBLIC_CLERK_PUBLISHABLE_KEY=\nCLERK_SECRET_KEY=\n\nNEXT_PUBLIC_CLERK_SIGN_IN_URL=/sign-in\nNEXT_PUBLIC_CLERK_SIGN_UP_URL=/sign-up\nNEXT_PUBLIC_CLERK_AFTER_SIGN_IN_URL=/onboarding\nNEXT_PUBLIC_CLERK_AFTER_SIGN_UP_URL=/onboarding\n\nGEMINI_API_KEY=\n\nARCJET_KEY=\n```\n\n---\n\n## 📸 Screenshots\n\n### 🏠 Home Page\n![Home](./screenshots/home.png)\n\n### 🔍 Filters\n![Filters](./screenshots/filters.png)\n\n### ❤️ Saved Cars\n![Saved](./screenshots/saved.png)\n\n### 🔐 Login Page\n![Login](./screenshots/login.png)\n\n---\n\n## 👨‍💻 Author\n\n* GitHub: https://github.com/satyavhan1\n
